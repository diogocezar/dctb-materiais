\subsection[Projeto OriAnt]{Projeto OriAnt}

\begin{frame}
    \frametitle{Projeto OriAnt}

    O projeto OriAnt foi dividido em três principais módulos:

    \begin{itemize}
        \item <1-> computação de feromônio;
        \item <2-> adaptação;
        \item <3-> administração.
    \end{itemize}
\end{frame}




\subsection{Módulo: Administração}

\begin{frame}
    \frametitle{Módulo: Administração}

    Dentre as funcionalidades disponíveis estão:

    \begin{itemize}

        \item <1-> inserir, alterar ou remover grupos;

        \item <2-> consultas sobre a freqüência dos grupos;

        \item <3-> alterar template;

        \item <4-> alteração dos parâmetros administrativos ($pA$);

        \item <5-> gerenciamento dos registros do banco de dados;

    \end{itemize}

\end{frame}
