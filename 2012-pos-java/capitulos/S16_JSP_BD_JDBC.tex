\section[JSP Banco de Dados (JDBC)]{JSP Banco de Dados (JDBC)}

\begin{frame}[c,allowframebreaks,fragile]
    \frametitle{JSP Banco de Dados (JDBC)}
    \begin{itemize}
      \item \textit{Java Database Connectivity} ou JDBC é um conjunto de classes
e interfaces (API) escritas em Java que fazem o envio de instruções SQL para
qualquer banco de dados relacional;
      \item Api de baixo nível e base para api's de alto nível;
      \item Amplia o que você pode fazer com Java;
      \item Possibilita o uso de bancos de dados já instalados; 
      \item Para cada banco de dados há um driver JDBC;
      
      \framebreak
      
      \item exemplo separado em arquivos:
      
      \begin{itemize}
	\item bd:
	\begin{itemize}
	 \item ConnectionBase - É a base para a conexão ao banco de dados;
	 \item DataBase - Herda a ConnectionBase e configura o banco de dados
baseado nas constantes definidas na classe ConfigSite;
	\end{itemize}
	\item config:
	\begin{itemize}
	  \item ConfigSite - Seta as configurações do banco e do site em
questão;
	\end{itemize}
	\item errors:
	\begin{itemize}
	  \item Error - Seta as mensagens de erro do sistema;
	\end{itemize}
	\item util:
	\begin{itemize}
	  \item Library - Biblioteca com funções do sistema;
	\end{itemize}
      \end{itemize}
    \end{itemize}
\end{frame}
