\begin{frame}[c,allowframebreaks]
    \frametitle{XHTML - Elementos de Lista}
    \begin{itemize}
      \item o XHTML dá suporte a três tipos de listas:
      \begin{itemize}
       \item ordenadas;
       \item não ordenadas; 
       \item de definição.
      \end{itemize}
      \item utiliza-se o marcador \texttt{<ul>} e \texttt{</ul>}, e dos
elementos \texttt{<li>} e \texttt{</li>};
      \item este tipo de lista pode através do atributo ``type'', apresentar
três tipos de marcadores:
      \begin{itemize}
       \item um circulo vazio (circle);
       \item um circulo cheio (disc); 
       \item um quadrado (square).
      \end{itemize}
      \item as listas ordenadas possuem elementos que são ``numerados'', o
marcador utilizado é o \texttt{<ol>} e \texttt{</ol>} e para definição de cada
sub-elemento são os \texttt{<li>} e \texttt{</li>}.
      \item o tipo de numeração pode ser alterado com o uso do atributo
``type'', o qual pode assumir os valores: 
      \begin{itemize}
       \item 1 (arábico);
       \item a, A (alfabético);
       \item i e I (romano).
      \end{itemize}
      \item podem ser utilizadas na definição de termos, o marcador geral são os
\texttt{<dl>} e \texttt{</dl>};
      \item os que identificam o texto a ser definido são os
\texttt{<dt>} e \texttt{</dt>} e a definição dos termos é feita através dos
\texttt{<dd>} \texttt{</dd>}.
    \end{itemize}
\end{frame}

    \frametitle{XHTML - Links}
    \begin{itemize}
      \item as ligações hipertexto são figuras ou trechos de textos que quando
clicados, ou acionados, redirecionam o navegador ou leitor XHTML à um outro
documento, ou a outro ponto do documento atual;
      \item outro atributo que pode ser utilizado pelo elemento \texttt{<a>} é o
