\section[Considerações Finais]{Considerações Finais}

\begin{frame}[c,allowframebreaks,fragile]
    \frametitle{Considerações Finais}
    \begin{itemize}
      \item por que não parar de estudar por aqui?
      \begin{itemize}
      	\item ambientes de produção: técnologia de ponta;
      	\item quanto menos trabalho e mais aproveitamento de código melhor!
      	\item tecnologias java para web já evoluíram;
      	\item por que aprendemos tudo isso?
      	\begin{itemize}
      		\item um bom profissional saber escovar \textit{bit}!
      		\item no final, tudo é \textit{servlet};
      		\item ajax independe da tecnologia;
      		\item programação \textit{front-side}: independe da tecnologia;
      	\end{itemize}
      \end{itemize}
      \framebreak
      \item e o que vem pela frente?
      \begin{itemize}
      	\item Não tem como fugir do MVC;
      	\item JSF: é uma tecnologia que incorpora característica de um framework MVC para \textit{web} e de um modelo de interfaces gráficas baseados em eventos;
      	\item JPA: é um \textit{framework} utilizado na camada de persistência, visando a produtividade (hibernate);
      	\item Facelets: assim como JSTL é uma nova maneira de se programar que se integra com JSF;
      \end{itemize}
      \item facilidades: as ferramentas geram código (cuidado!);
      \item vamos dar uma olhada em uma aplicação de exemplo JSF; (só uma olhada!)
    \end{itemize}
\end{frame}
